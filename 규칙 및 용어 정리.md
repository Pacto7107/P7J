## 용어
1. 이용자 : P7J를 사용하는 사람
2. 레이팅 : P7J에서 주어지는 공식으로 계산되는 각 이용자의 점수
3. 티어 : 레이팅이 어떤 점수에 도달했을 때 얻는 것
4. 레이팅 표 : 각 이용자의 레이팅과 티어, 티어의 간격이 써져있는 표 또는 종이
5. 리더 : 레이팅 표를 각 이용자의 레이팅에 맞게 수정하고, 이용자의 행동을 판단해 알맞은 처리를 하는 사람
6. 이벤트 : 리더가 원할 때 어느 적당한 기간동안 레이팅 변동폭을 변경하거나, 모두에게 레이팅을 올리는 것 등이다.

## 규칙
### 1장 총강
1. 이 규칙은 P7J의 규칙이다.
2. 이 규칙의 모든 용어는 상단에 정의되어있지 않는 한 <표준국어대사전> 을 따른다. 상단에 정의되어 있을 경우에는 그 정의를 따른다.
3. P7J의 이용자이면서 이 규칙을 지키지 않았을 경우 제재될 수 있다.

### 2장 레이팅 계산법
1. 2장에서 레이팅이 늘어나거나 깎이는 것은 별도 언급이 없을 때 이벤트의 영향을 **받는다**
2. 노랭크전은 레이팅에 포함되지 않는다.  
3. 랭크전에서 진 사람의 레이팅은 1 깎인다.
4. 랭크전에서 이긴 사람의 레이팅은 1 는다.
5. 왕중왕전에서 어떤 한 사람이 그 왕중왕전 한 판에서 참여한 라운드의 개수를 `n` 이라 하고, 그 왕중왕전에 참여한 이용자의 명수를 `all` 이라 할 때 그 사람의 레이팅 변동폭은 `(n <= int(all / 2)) ? n - int(all / 2) - 1 : n` 이다.

### 4장 제재
1. 제재의 종류는 다음 표와 같다.

| 이름 | 실행 내용 |
| ------ | --------- |
| `-1!` | 이용자의 레이팅을 -1 한다. |
| `경고` | 3번 받으면 `-1` 이다. |
| `A` | 3번 받으면 `경고` 이다. |
| `B` | 3번 받으면 `A` 이다. |
| `도미노!` | 이용자의 레이팅을 `-10` 한다. |
2. 제재는 이벤트의 영향을 받지 **않는다**.
3. 리더는 다른 부적절한 행동을 하거나 규칙을 지키지 않은 사용자를 발견하면 제재할 수 있다.

### 5장 젠가
1. 동맹은 가능하다.
2. 이상젠가는 기본적으로 노랭크전이다.
3. 젠가 할 때는 기본적으로 한 턴에 최대 2개의 젠가 조각만 뽑을 수 있다.

#### 5-1장 왕중왕전 규칙
1. 2명이서 왕중왕전을 하지 않는다. 질 때 `-1` 이길 때 `2` 이다.
2. 3명이서 왕중왕전을 하지 않는다. 랭크전을 `n` 번 해라.
